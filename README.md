### 🚀 O mnie / About Me

> *"Kombinacja szybkości Rust i elastyczności nowoczesnego oprogramowania."*

- 🦀 **Rust & Low-Level**: Programowanie niskopoziomowe, optymalizacja pamięci i pisanie wydajnych aplikacji w **Rust**.
- 🎮 **Minecraft Engine & Modding**: Tworzenie zaawansowanych autorskich silników, pluginów i trybów rozgrywki.
- ⚡ **Backend & API**: Projektowanie mikroserwisów, wydajnych REST API oraz integracja z bazami danych.
- 🐧 **Linux & DevOps**: Administracja systemami Linux, konteneryzacja w Dockerze i automatyzacja workflow.
- 🌱 **Ciągły Rozwój**: Eksperymentowanie z architekturą oprogramowania, algorytmami i nowymi technologiami.
